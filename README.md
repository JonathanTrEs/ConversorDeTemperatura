Descripcion: En esta practica desarrollaremos el código javascript para un conversor de temperatura, de grados
Celsius a Fahrenheit y viceversa

La estructura de nuestra practica es la siguente:
.
├── global.css
├── index.html
├── temperature.js
└── tests
    ├── chai.js
    ├── index.html
    ├── mocha.css
    ├── mocha.js
    └── tests.js

En el directorio principal tenemos los siguientes ficheros:
      - global.css: este fichero contiene el estilo de nuesto .html.
      - index.html: fichero .html principal desde el cual se ejecuta nuestro código javascript.
      - temperature.js: fichero que contiene el código javascript para la conversión de la temperatura.
      
En el directorio tests tenemos los siguientes ficheros:
      - chai.js: fichero que nos hemos bajado del correspondiente repositorio de internet.
      - index.html: fichero parecido al index del directorio principal, pero donde hemo añadido las llamadas a las pruebas.
      - mocha.css: fichero con el estilo de las pruebas.
      - mocha.js: fichero generado automaticamente por mocha.
      - tests.js: fichero donde ponemos el código de las pruebas para ejecutarlas.
